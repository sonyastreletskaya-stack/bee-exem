Политика обработки персональных данных

Настоящая политика описывает порядок обработки данных пользователей сайта ПГЭ-2026.

1. Какие данные собираются

При регистрации пользователь указывает ник, электронную почту и пароль. Также в системе сохраняются результаты прохождения тестов, время выполнения заданий и текст сочинения, отправленный пользователем.

2. Для чего используются данные

Данные используются для регистрации участника, входа в личный кабинет, сохранения результатов экзаменационных испытаний и проверки сочинения.

3. Где хранятся данные

Данные хранятся в базе данных Supabase. Пароли обрабатываются средствами системы авторизации и не отображаются администрации сайта в открытом виде.

4. Доступ к данным

Пользователь видит свои результаты после входа в личный кабинет. Администрация проекта может видеть ник, почту, результаты тестов и отправленные сочинения для проверки и технической поддержки.

5. Срок хранения

Данные хранятся до окончания проведения ПГЭ или до удаления данных администрацией проекта.

6. Согласие

Регистрируясь на сайте, пользователь подтверждает согласие на обработку указанных данных.

7. Контакты

По вопросам удаления или изменения данных пользователь может обратиться к администрации проекта.

Это базовая версия политики для проекта. Для полностью юридически строгого документа текст нужно отдельно проверить под вашу конкретную ситуацию.
